1 Topics
2 ======
3 
4 Topics is the term used to reference the metadata property that `Project Open Data <https://project-open-data.cio.gov>`_ calls "theme".
5 The `theme <https://project-open-data.cio.gov/v1.1/schema/#theme>`_ is the main thematic category of the dataset.
6 
7 DKAN will create a *data* node for each theme specified in the datasets.
8 
9 When the value of themes change or become outdated, the corresponding data node will be removed by the **orphan_reference_processor** queue task.
10 If you prefer to run it manually, you may do so with:
11 
12 .. code-block::
13 
14  drush queue-run orphan_reference_processor
