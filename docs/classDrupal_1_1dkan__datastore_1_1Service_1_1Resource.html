Drupal\dkan_datastore\Service\Resource::remove ( )

Remove the resource object.

Resource objects are dynamic, so they themselves do not need to be removed. However, to create a resource we use the filefetcher. Filefetcher states are stored, and once fetching is finished a file exists in the file system. We need to clean up both of those things.
